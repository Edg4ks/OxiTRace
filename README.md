## Overview

**Oxitrace** is a modern HTTP analysis and inspection toolkit focused on performance, clarity, and low-level visibility into HTTP communications.

Designed for developers, security researchers, and network engineers, Oxitrace provides detailed request and response analysis directly from the terminal with the speed and reliability of Rust.

---

## Features

- HTTP request & response inspection
- Header and metadata analysis
- Response fingerprinting
- Latency measurement
- Raw request visualization
- Lightweight and fast CLI workflow
- Modular Rust architecture
- Security research oriented
